Privacy Policy

This extension, TextMeter, does not collect, store or transmit any personal data or usage information.

All processing is done locally on your device. No information is shared with third parties.

The extension requests the following permissions solely for its core functionality:

  • activeTab: to access the content of the active tab when you trigger the extension.
  • scripting: to inject and execute code that reads selected text.
  • contextMenus: to add an action to the right-click menu.
  • host access (<all_urls>): to allow functionality across all websites.

No data leaves your browser at any point.

If you have any questions or concerns, please contact the developer via the Chrome Web Store feedback form.
